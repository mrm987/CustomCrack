# C.Crack (Custom Crack)

[crack.wrtn.ai](https://crack.wrtn.ai/) 커스텀 래퍼 데스크톱 앱.  
크랙의 UI를 개인 취향에 맞게 커스터마이징할 수 있습니다.

## 설치

[Releases](https://github.com/mrm987/CustomCrack/releases)에서 최신 `.exe` 파일을 다운로드하여 실행하세요.  
별도 설치 과정 없이 바로 실행됩니다. (Portable)

## 기능

### 보더리스 윈도우
- 기본 윈도우 타이틀바를 제거하고, 크랙 헤더에 최소화/최대화/닫기 버튼을 통합
- 크랙 헤더를 드래그하여 창 이동 가능

### 커스텀 메뉴 (우측 하단 톱니바퀴 버튼)

| 기능 | 기본 단축키 | 설명 |
|---|---|---|
| **전체화면** | `F11` | 전체화면 모드 토글 |
| **몰입모드** | `F10` | 헤더, 사이드바, 서브헤더를 숨기고 채팅 영역만 표시 |
| **버튼 숨기기** | `F9` | 커스텀 메뉴 버튼을 투명하게 숨김 (호버 시 표시) |
| **상태창 스타일** | - | 코드블록 디자인을 기본(헤더 있음)/클린(보더리스) 모드로 전환 |
| **테마** | - | 라이트/다크 모드 전환 |

### 편의 기능

| 기능 | 기본 단축키 | 설명 |
|---|---|---|
| **유저노트** | `F8` | 현재 채팅방의 유저노트 팝업을 바로 열기 |
| **대화 프로필** | `F7` | 현재 채팅방의 대화 프로필 팝업을 바로 열기 |
| **요약 메모리** | `F6` | 현재 채팅방의 요약 메모리 팝업을 바로 열기 |
| **크래커 표시** | - | 헤더에 크래커 잔액을 실시간 표시 (채팅 시 자동 갱신) |

### 채팅 커스텀

| 기능 | 설명 |
|---|---|
| **채팅 너비** | 채팅 영역의 가로 너비를 50%~250% 범위에서 조절 |
| **폰트 크기** | 채팅 메시지의 폰트 크기를 50%~200% 범위에서 조절 |

### 채팅 입력 히스토리
- **방향키 위/아래**로 이전에 입력한 메시지를 불러올 수 있음
- **ESC** → 히스토리 탐색 취소
- 최대 100개 저장, 앱 재시작 후에도 유지

### 단축키 커스터마이즈
- 각 버튼의 단축키 라벨을 클릭하면 변경 가능
- `Ctrl+키`, `Shift+키`, `Alt+키` 등 조합키 지원
- 중복 단축키 자동 감지 및 차단

### 기타 단축키

| 단축키 | 설명 |
|---|---|
| `F5` | 페이지 새로고침 |
| `F12` | 개발자 도구 열기/닫기 |
| `Ctrl+Shift+R` | 커스텀 CSS/JS 핫 리로드 |

## 설정 저장
모든 설정은 로컬 브라우저(localStorage)에 자동 저장되어 다음 실행 시 복원됩니다.

## 개발

```bash
# 의존성 설치
npm install

# 개발 실행
npm start

# 빌드
npm run build
```
